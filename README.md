# CloudPulse - Product Showcase Landing Page

A modern landing page for CloudPulse, an infrastructure monitoring SaaS product.

**Live Demo:** https://arpine11.github.io/landing_page_vibe_coding_project/

## Features

### v1 — Core Landing Page
- Hero section with product tagline and CTA
- Product showcase with 4 feature cards
- Benefits section with icons
- Customer testimonials
- 3-tier pricing section (Starter / Pro / Enterprise)
- Responsive footer with social links

### Merged — UX Enhancements
- Sticky navbar that stays fixed on scroll
- Smooth scrolling for anchor navigation
- Active section highlight in the navbar
- CTA modal with signup form (name + email)
- Client-side form validation (required fields, email format)
- Modal close via close button, backdrop click, or Escape key

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/arpine11/landing_page_vibe_coding_project.git
   ```
2. Open `index.html` in your browser.

No build tools required — it's plain HTML, CSS, and JavaScript.
